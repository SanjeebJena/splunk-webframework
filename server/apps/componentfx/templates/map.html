Map Views

The GoogleMap view binds a map to a search that interprets geographic data and displays it on a map. Data with lat and lng fields are used to populate map markers according to location.

Example

In this example, we use the {% googlemap %} tag to create the GoogleMap view, which is bound to a search that includes latitude and longitude values. Search results (markers) are automatically displayed at their locations on the map.

{% googlemap id="example-map" managerid="geosearch" %}

var GoogleMapView = require("splunkjs/mvc/googlemapview");
new GoogleMapView({
    name: "example-map",
    managerid: "geosearch",
    el: $("#divToHangOn")
}).render();

var SearchManager = require("splunkjs/mvc/searchmanager");
new SearchManager({
    name: "geosearch",
    search: "| inputlookup earthquakes.csv | rename Lat as lat Lon as lng | stats count by lat, lng"
});

Options

    tileSource: Tiles to use in map render. Can be openStreetMap or splunk. Defaults to splunk.
