Cascading Forms

In this case, we have two form select views, and the choice of one cascades into the other. When you select an index, we run a search to find which sourcetypes are available on that index. Finally, when you click the Search button, we use both selections to run the final search.

Note how we are using the same approach as when we had form inputs feeding into searches directly: there is no difference between a search that populates a form and a search that populates a view.
