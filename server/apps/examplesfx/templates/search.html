Recreating the Search Timeline UI

This example shows how you can use the components of the Application Framework to recreate portions of the Splunk Search Timeline UI. Without any custom code, we can put the views and managers we want on the page.

Another benefit is that this is still just an HTML/JS/CSS page: we can add custom HTML (such as this blurb), or even add custom interaction using JavaScript.
