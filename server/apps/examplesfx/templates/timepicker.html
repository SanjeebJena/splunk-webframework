Time Pickers

Using a time picker is a common way to control the time bounds of a search. This example shows you can can easily set up a time picker, without any custom code. This example uses Django template tags, but the same can easily be achieved in JavaScript.

Multiple time pickers can exist simultanenously on the page, controlling different searches (or even the same search).

Time Picker 1

{% timepicker id="timepicker" managerid="simplesearch1" %} {% single id="single1" managerid="simplesearch1" beforeLabel="_internal event Count: " %}
