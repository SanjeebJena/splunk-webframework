Splunk App Framework Quick Start

Use the sample data

For this Quick Start and our other sample apps, we included some sample geo-tagged data so that you can experiment with the different features of the new Splunk App Framework—even if you don't have any data in Splunk yet.

Where is the sample data?

The sample data is ready to go, you don't need to do anything. You can even use it with your other apps if you want to. Just copy the appfx.demo.dataset.csv file from $HOMEDIR/server/apps/quickstartfx/splunkd/lookups to your app's directory under $HOMEDIR/server/apps/your_app/splunkd/lookups.

Search the sample data

To search the sample data, use the inputlookup command in your search query. For example, this search query lists every event in the data set:

search = " | inputlookup appfx.demo.dataset.csv"

This example returns the count of events:

search = " | inputlookup appfx.demo.dataset.csv | stats count"
