Splunk App Framework Quick Start

Add a search

Search is a core feature of the Splunk App Framework. All of the framework UI controls can be bound to a search in various ways. Some controls are populated by searches, some modify the searches they are bound to, and some do both.

Each search is represented by a search context, which manages the query, properties, operation, and lifetime of an individual search job. You can create search contexts using Django template tags or JavaScript. Search contexts can correspond to a saved search, or a search job you create on the fly in code or dynamically with UI controls.

Creating a search context

A search context can be input dynamically using a SearchBar control or hardcoded into a framework app.
