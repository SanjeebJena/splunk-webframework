Splunk App Framework Quick Start

Add a search bar to your app

A quick and easy way to add dynamic search functionality to your app is by using a SearchBar control.

Add a search bar

The search bar is a standard control that includes a text box for entering search queries and (optionally) a time picker that allows the user to change the search timeline. The ID of the search context to bind to is set in the SearchBar properties. When a new search query or timeline is submitted, the search context is updated and any other controls that are bound to the search context are notified.

Here is a basic search bar (although this doesn't do anything yet):

{% searchbar id="example search bar" contextid="example-search" %} {% search id="example-search" %}

Create the control is by adding the searchbar tag to your HTML template, including the ID of the search context in the contextid property.

{% searchbar id="example search bar" contextid="example-search" %}

{% search id="example-search" %}

Search bars can also be bound to and modify an existing hard-coded search.
