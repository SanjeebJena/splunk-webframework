Splunk App Framework Quick Start

Add a search

You can include a hard-coded search or any saved search from your Splunk instance to your app.

Adding a search context

Search contexts can be embedded in your framework app. Searches are run against your Splunk instance. The data that is returned is available to any control that is bound to the search context. To include an embedded search context with a static search query, create a SearchContext with a query defined in the "search" property:

{% search id="example-search" search="search index=_internal | stats count" %}

Other parameters such as "earliest_time", "latest_time", etc. can also be set through properties.

Adding a saved search context

To add a saved search, create a SavedSearchContext. You'll need to provide the name of the saved search in the searchname property. To bind controls to this context, provide the ID of this SavedSearchContext context in the "contextid" property of the control.

{% savedsearchcontext id="saved-search" searchname="example-saved-search" %}
