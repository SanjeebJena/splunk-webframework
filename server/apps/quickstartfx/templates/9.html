Splunk App Framework Quick Start

Next steps

Now you're ready to create your own apps using the Splunk App Framework. And if you ever want to run this Quick Start again, you can find it as the Quick Start app on the Home page of the Splunk App Framework.

Where is the sample data for this Quick Start?

For this Quick Start and our other sample apps, we included some sample geo-tagged data. If you would like to use the data in your other apps, just copy the appfx.demo.dataset.csv file from $APPFX_HOME/server/apps/quickstartfx/splunkd/lookups to your app's directory under $APPFX_HOME/server/apps/your_app/splunkd/lookups.

Search the sample data

To search the sample data, use the inputlookup command in your search query. For example, this search query lists the count of events in the data set:

| inputlookup appfx.demo.dataset.csv | stats count
